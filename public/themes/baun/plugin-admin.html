We currently support all major credit cards including Visa, Mastercard, American Express and Discover. We also accept PayPal payments.

How do updates work?

As the Admin plugin is a composer package, updates will always be free. However you will need an active license to use the plugin.

How do licenses work?

When you install the plugin you will be asked to add your license key (which you will recieve in your receipt email) to the plugin config file. The plugin will not work without a valid license. Licenses last for one year.

What happens when my license expires?

When your license expires the plugin will no longer work, and you will need to renew your license (at a discounted rate).

Can I run additional websites with a single license?

No. A license is valid for a single plugin installation, running on a single domain. If you plan to add additional installations you need to purchase licenses for each of those installations.
